# Portfolio Website

This is a personal portfolio website built with HTML, CSS, and JavaScript. It showcases projects, work experience, skills, and contact information in a modern, responsive, and visually appealing format.

## Features

- **About Section:** Introduction, profile picture, and quick stats.
- **Portfolio:** Project cards with GitHub and live demo links.
- **Work Experience:** Timeline of internships and jobs.
- **Skills:** Technical skills and proficiency.
- **Contact Form:** AJAX-powered form using FormSubmit (no page reload, success/error message auto-hides after 10 seconds).
- **Dark Theme:** Default dark mode for a modern look.
- **Favicon:** Custom favicon using your profile image.
- **Responsive Design:** Works on all devices.

## Technologies Used

- HTML5
- CSS3 (custom and Swiper.js for sliders)
- JavaScript (vanilla, with AJAX for contact form)
- [FormSubmit](https://formsubmit.co/) for contact form backend
- [Unicons](https://iconscout.com/unicons) for icons
